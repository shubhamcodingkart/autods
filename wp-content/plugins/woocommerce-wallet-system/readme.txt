== Description ==

Woocommerce Wallet System minimises the site customer(s) load of using hard cash or debit/credit card each and every time making a new transaction by
one click payment using wallet system. Admin can perform manual transaction in customer(s) wallet and can restrict minimum and maximum wallet credit as well as debit amount respectively. Admin can provide cashbacks to customer using custom cashback rules. Customer can entertain wallet to wallet transaction as well.

== Installation ==

1. Install the Woocommerce Wallet System
2. Activate the plugin
3. Go to the Customer Wallet
